# DragAndDropTest.java - Drag & Drop amb la classe Actions

---
El drag & drop (arrossegar i deixar anar) és una interacció avançada que no es pot realitzar amb un simple `click()`. Selenium ho gestiona a través de la classe `Actions`, que permet encadenar gestos complexos del ratolí. Aquest test treballa contra [compendiumdev.co.uk — GUI User Interactions](http://compendiumdev.co.uk/selenium/testpages/gui_user_interactions.html).

___

### 📄 Codi del test

```java
DragAndDropTest.java — Drag & Drop amb la classe Actionspublic class DragAndDropTest extends BaseTest {

    @Test
    public void testApp() throws InterruptedException {

        driver.navigate().to(
            "http://compendiumdev.co.uk/selenium/testpages/gui_user_interactions.html");

        // Localitzar els elements arrossegables i les zones de destí
        WebElement draggable1 = driver.findElement(
            By.xpath(".//*[@class='draganddrops']/div/div[1]"));
        WebElement draggable2 = driver.findElement(By.id("draggable2"));
        WebElement droppable1 = driver.findElement(By.id("droppable1"));
        WebElement droppable2 = driver.findElement(By.id("droppable2"));

        // Crear l'objecte Actions
        Actions dragAndDrop = new Actions(driver);

        // Arrossegar draggable1 fins a droppable1
        dragAndDrop.dragAndDrop(draggable1, droppable1).perform();
        Thread.sleep(4000);

        // Arrossegar draggable2 fins a droppable2
        dragAndDrop.dragAndDrop(draggable2, droppable2).perform();
        Thread.sleep(4000);
    }
}
```

___

### 🔍 Anàlisi del test

#### El mètode `dragAndDrop(origen, destí)`

És el mètode més directe per al drag & drop. Rep l'element a arrossegar i l'element destí on deixar-lo anar. Internament executa la seqüència completa: clic sostingut → moure → deixar anar.

```java
Actions actions = new Actions(driver);
actions.dragAndDrop(elementOrigen, elementDesti).perform();
```

___

### 🧰 Les dues formes de fer drag & drop

El test inclou en comentaris una segona forma equivalent, més explícita i amb més control:

#### Forma 1 — `dragAndDrop()` (simplificada)

```scss
actions.dragAndDrop(draggable2, droppable1).perform();
```

#### Forma 2 — `clickAndHold + moveToElement + release` (explícita)

```scss
actions
    .clickAndHold(draggable2)   // Clic i mantenir premut
    .moveToElement(droppable1)  // Moure fins al destí
    .release()                  // Deixar anar el botó del ratolí
    .build()
    .perform();
```

|  | `dragAndDrop()` | `clickAndHold + move + release` |
|----------------|------------------------------------|-------------------------------------------------|
| **Codi** | Més concís | Més verbós |
| **Control** | Limitat | Total — permet pauses intermèdies |
| **Compatibilitat** | Pot fallar en alguns frameworks JS | Més compatible amb llibreries com React/Angular |
| **Ús recomanat** | HTML5 natiu drag & drop simple | Drag & drop amb lògica JavaScript complexa |

___

### 🧩 La classe `Actions` — resum de mètodes

| Mètode | Descripció |
|----------------------------|-----------------------------------------------------------|
| `dragAndDrop(origen, destí)` | Arrossega l'element origen i el deixa anar al destí |
| `clickAndHold(element)` | Fa clic i manté el botó premut |
| `moveToElement(element)` | Mou el cursor fins a l'element indicat |
| `release()` | Deixa anar el botó del ratolí |
| `click(element)` | Fa clic sobre un element |
| `doubleClick(element)` | Fa doble clic sobre un element |
| `contextClick(element)` | Fa clic dret sobre un element |
| `sendKeys(Keys.X)` | Envia una tecla especial al focus actual |
| `build()` | Construeix la seqüència d'accions (opcional a Selenium 4) |
| `perform()` | Executa la seqüència al navegador |
