# InterrogationTest.java - Inspecció del DOM | Odissea

---
Un cop Selenium ha carregat una pàgina, podem **interrogar el driver** per obtenir informació sobre l'estat actual del navegador: el títol de la pàgina, la URL actual, el codi font HTML complet, o propietats de qualsevol element. Aquest test mostra els mètodes d'inspecció més útils.

___

### 📄 Codi del test

```csharp
public class InterrogationTest extends BaseTest {

    @Test
    public void testNavigation() throws Exception {

        driver.get("https://www.duckduckgo.com/"); // Naveguem a DuckDuckGo

        System.out.println("---------------------");
        System.out.println(driver.getPageSource()); // HTML complet de la pàgina
        System.out.println("---------------------");
        System.out.println(driver.getTitle());      // Títol de la pestanya
        System.out.println("---------------------");
        System.out.println(driver.getCurrentUrl()); // URL actual del navegador
        System.out.println("---------------------");
    }
}
```

___

### 🔍 Mètodes d'inspecció del driver

#### `driver.getTitle()` — Títol de la pàgina

Retorna el text de l'element `<title>` de la pàgina actual. Molt útil per verificar que hem navegat a la pàgina correcta.

```csharp
String titol = driver.getTitle();
System.out.println(titol); // "DuckDuckGo — Privacy, simplified."

// Ús en asserció
assertEquals(driver.getTitle(), "DuckDuckGo — Privacy, simplified.");
```

___

#### `driver.getCurrentUrl()` — URL actual

Retorna la URL completa que mostra el navegador en aquell moment. Útil per verificar redireccions o que una acció ha portat a la pàgina esperada.

```csharp
String url = driver.getCurrentUrl();
System.out.println(url); // "https://www.duckduckgo.com/"

// Ús en asserció
assertTrue(driver.getCurrentUrl().contains("duckduckgo"));
```

___

#### `driver.getPageSource()` — Codi font HTML

Retorna l'**HTML complet** de la pàgina tal com l'ha rebut el navegador (incloent-hi els canvis dinàmics del DOM generats per JavaScript). Útil per a la depuració o per verificar la presència de text a la pàgina.

```php-template
String html = driver.getPageSource();
System.out.println(html); // <!DOCTYPE html><html>...

// Verificar que un text apareix en algun lloc de la pàgina
assertTrue(driver.getPageSource().contains("Privacy"));
```

> 💡 **Tip:** `getPageSource()` retorna el DOM _en el moment de la crida_, incloent-hi qualsevol modificació que JavaScript hagi fet sobre l'HTML original. No és l'HTML estàtic del servidor, sinó l'estat actual del DOM.

___

### 🔎 Inspecció d'elements individuals

A més d'interrogar el driver, també podem obtenir informació de qualsevol **element concret** de la pàgina:

```java
InterrogationTest.java — Inspecció del DOMWebElement element = driver.findElement(By.id("search_form_input_homepage"));

// Text visible de l'element
String text = element.getText();

// Valor d'un atribut HTML
String placeholder = element.getAttribute("placeholder");
String href        = element.getAttribute("href");
String classeCSS   = element.getAttribute("class");

// Valor d'una propietat CSS
String color       = element.getCssValue("color");
String font        = element.getCssValue("font-size");

// Estat de l'element
boolean esVisible  = element.isDisplayed(); // Està visible a la pantalla?
boolean estaActiu  = element.isEnabled();   // Està habilitat (no disabled)?
boolean marcat     = element.isSelected();  // Està marcat? (checkbox/radio)

// Posició i mida
Point     posicio  = element.getLocation(); // Coordenades (x, y) a la pàgina
Dimension mida     = element.getSize();     // Amplada i alçada en píxels
```

___

### 📊 Resum de mètodes d'inspecció

| Mètode | Sobre | Retorna | Ús típic |
|------------------------|------------|---------|--------------------------------------------|
| `driver.getTitle()` | Driver | `String` | Verificar que estem a la pàgina correcta |
| `driver.getCurrentUrl()` | Driver | `String` | Verificar redireccions i URLs |
| `driver.getPageSource()` | Driver | `String` | Depuració, cercar text al DOM |
| `element.getText()` | WebElement | `String` | Llegir el text visible d'un element |
| `element.getAttribute()` | WebElement | `String` | Llegir atributs HTML (`href`, `class`, `value`…) |
| `element.getCssValue()` | WebElement | `String` | Verificar estils CSS aplicats |
| `element.isDisplayed()` | WebElement | `boolean` | Comprovar la visibilitat de l'element |
| `element.isEnabled()` | WebElement | `boolean` | Comprovar si l'element està habilitat |
| `element.isSelected()` | WebElement | `boolean` | Comprovar si un checkbox/radio està marcat |
