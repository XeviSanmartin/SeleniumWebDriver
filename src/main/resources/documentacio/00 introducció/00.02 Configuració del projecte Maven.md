# Configuració del projecte Maven | Odissea

---
El projecte utilitza **Apache Maven** com a gestor de dependències i construcció. El fitxer `pom.xml` defineix tot el que necessita el projecte per compilar i executar els tests.

___

### 📦 Dependències principals

```php-template
Configuració del projecte Maven — pom.xml<dependencies>

  <!-- TestNG: framework de testing -->
  <dependency>
    <groupId>org.testng</groupId>
    <artifactId>testng</artifactId>
    <version>7.10.2</version>
  </dependency>

  <!-- Selenium WebDriver (llibreria principal) -->
  <dependency>
    <groupId>org.seleniumhq.selenium</groupId>
    <artifactId>selenium-java</artifactId>
    <version>4.43.0</version>
  </dependency>

  <!-- Driver específic de Firefox -->
  <dependency>
    <groupId>org.seleniumhq.selenium</groupId>
    <artifactId>selenium-firefox-driver</artifactId>
    <version>4.43.0</version>
  </dependency>

  <!-- Driver específic de Chrome -->
  <dependency>
    <groupId>org.seleniumhq.selenium</groupId>
    <artifactId>selenium-chrome-driver</artifactId>
    <version>4.43.0</version>
  </dependency>

  <!-- Hamcrest: llibreria d'assercions -->
  <dependency>
    <groupId>org.hamcrest</groupId>
    <artifactId>hamcrest-core</artifactId>
    <version>2.2</version>
  </dependency>

</dependencies>
```

___

### 🔧 Plugin d'execució: Maven Surefire

El plugin `maven-surefire-plugin` és l'encarregat d'executar els tests de TestNG. Està configurat per llegir el fitxer `src/main/resources/suite.xml`, que defineix quins tests s'executen i en quin ordre.

```php-template
<plugin>
  <groupId>org.apache.maven.plugins</groupId>
  <artifactId>maven-surefire-plugin</artifactId>
  <version>3.1.2</version>
  <configuration>
    <suiteXmlFiles>
      <suiteXmlFile>src/main/resources/suite.xml</suiteXmlFile>
    </suiteXmlFiles>
  </configuration>
</plugin>
```

___

### ▶️ Comandes Maven essencials

| Comanda | Què fa? |
|--------------------------------|----------------------------------------------------------------|
| `mvn test` | Executa tots els tests definits a `suite.xml` |
| `mvn test -Dbrowser=firefox` | Executa els tests utilitzant Firefox |
| `mvn test -Dtest=DuckDuckGoTest` | Executa únicament la classe `DuckDuckGoTest` |
| `mvn clean test` | Neteja els compilats anteriors i executa els tests des de zero |

___

> **💡 Tip:** A partir de Selenium 4.6+, el gestor de drivers està integrat a la pròpia llibreria (_Selenium Manager_). En molts casos ja no cal descarregar manualment `chromedriver` ni `geckodriver` — Maven ho gestiona sol.
