# InfiniteScrollTest.java - Scroll infinit amb JavascriptExecutor 


---
El scroll infinit és un patró de UI en el qual el contingut es carrega automàticament a mesura que l'usuari arriba al final de la pàgina. Selenium no té un mètode natiu per fer scroll, però podem usar **`JavascriptExecutor`** per executar JavaScript directament al navegador i desplaçar-nos fins al final del document. Treballa contra [the-internet.herokuapp.com/infinite\_scroll](http://the-internet.herokuapp.com/infinite_scroll).

___

### 📄 Codi del test

```java
public class InfiniteScrollTest extends BaseTest {

    @Test
    public void infiniteScrollTest() throws InterruptedException {
        driver.navigate().to("http://the-internet.herokuapp.com/infinite_scroll");

        // Fer scroll fins al final de la pàgina 5 vegades
        // esperant 2,5 segons entre cada scroll perquè carregui el nou contingut
        for (int i = 0; i < 5; i++) {
            jse.executeScript("window.scrollTo(0, document.body.scrollHeight);");
            Thread.sleep(2500);
        }
    }
}
```

___

### 🔍 Anàlisi: `JavascriptExecutor`

`JavascriptExecutor` és una interfície de Selenium que permet executar codi JavaScript arbitrari en el context del navegador. A `BaseTest` ja està instanciat com a `jse`:

```java
// A BaseTest.java (configuració base)
JavascriptExecutor jse = (JavascriptExecutor) driver;
```

#### La comanda de scroll

```bash
jse.executeScript("window.scrollTo(0, document.body.scrollHeight);");
```

| Paràmetre | Valor | Significat |
|-----------------------|----------------------------|-----------------------------------------------------|
| `window.scrollTo(x, y)` | — | Desplaça la finestra a les coordenades (x, y) |
| x | `0` | Sense desplaçament horitzontal |
| y | `document.body.scrollHeight` | L'alçada total del document (el final de la pàgina) |

___

### 🧰 Altres comandes útils amb `JavascriptExecutor`

```javascript
InfiniteScrollTest.java — Scroll infinit amb JavascriptExecutor// Scroll fins al final de la pàgina
jse.executeScript("window.scrollTo(0, document.body.scrollHeight);");

// Scroll fins a l'inici de la pàgina
jse.executeScript("window.scrollTo(0, 0);");

// Scroll fins a un element concret
WebElement element = driver.findElement(By.id("el-meu-element"));
jse.executeScript("arguments[0].scrollIntoView(true);", element);

// Fer clic en un element via JS (útil quan el clic normal falla)
jse.executeScript("arguments[0].click();", element);

// Canviar el valor d'un input via JS
jse.executeScript("arguments[0].value = 'nou valor';", inputElement);

// Obtenir el títol de la pàgina
String titol = (String) jse.executeScript("return document.title;");

// Obtenir la URL actual
String url = (String) jse.executeScript("return window.location.href;");
```

> 💡 **Quan usar JavascriptExecutor:** Usa'l com a últim recurs quan els mètodes natius de Selenium no funcionen (elements no clicables per superposició, scroll en contenidors específics, manipulació directa del DOM). En tests normals, sempre és preferible usar els mètodes natius de Selenium.
