#  CheckBoxesTest.java - Gestió de checkboxes 

---
Els checkboxes són un dels controls més comuns en formularis web. Selenium permet verificar el seu estat (marcat o no) i interactuar-hi mitjançant `click()` i el mètode `getAttribute("checked")`. Aquest test treballa contra la pàgina de pràctica [the-internet.herokuapp.com/checkboxes](https://the-internet.herokuapp.com/checkboxes).

___

### 📄 Codi del test

```java
CheckBoxesTest.java — Gestió de checkboxespublic class CheckBoxesTest extends BaseTest {

    @Test
    public void testApp() throws InterruptedException {

        // 1. Navegar fins a la pàgina de checkboxes
        driver.navigate().to("https://the-internet.herokuapp.com/checkboxes");

        // 2. Verificar que checkbox2 està marcat
        WebElement checkbox2 = driver.findElement(
            By.xpath(".//*[@id='checkboxes']/input[2]"));
        assertTrue(checkbox2.getAttribute("checked").equals("true"));

        // 2.1 Verificar que checkbox1 NO està marcat
        WebElement checkbox1 = driver.findElement(
            By.xpath(".//*[@id='checkboxes']/input[1]"));
        assertNull(checkbox1.getAttribute("checked"));

        // 3. Marcar checkbox1
        checkbox1.click();

        // 4. Verificar que ha quedat marcat
        assertTrue(checkbox1.getAttribute("checked").equals("true"));
    }
}
```

___

### 🔍 Anàlisi pas a pas

#### Pas 1 — Navegar a la pàgina

```scss
driver.navigate().to("https://the-internet.herokuapp.com/checkboxes");
```

Obre la pàgina de pràctica que conté dos checkboxes: el primer desmarcat i el segon marcat per defecte.

___

#### Pas 2 — Verificar que checkbox2 està marcat

```vbnet
WebElement checkbox2 = driver.findElement(
    By.xpath(".//*[@id='checkboxes']/input[2]"));
assertTrue(checkbox2.getAttribute("checked").equals("true"));
```

Per saber si un checkbox està marcat, llegim el seu atribut HTML `checked`:

-   Si està **marcat** → `getAttribute("checked")` retorna `"true"`
-   Si està **desmarcat** → `getAttribute("checked")` retorna `null`

> 💡 **Important:** L'atribut `checked` no retorna `"false"` quan el checkbox està desmarcat — retorna directament `null`. Per això l'asserció per a "no marcat" utilitza `assertNull()` i no `assertEquals(..., "false")`.

___

#### Pas 2.1 — Verificar que checkbox1 NO està marcat

```java
WebElement checkbox1 = driver.findElement(
    By.xpath(".//*[@id='checkboxes']/input[1]"));
assertNull(checkbox1.getAttribute("checked"));
```

___

#### Passos 3 i 4 — Marcar el checkbox i verificar

```less
checkbox1.click();
assertTrue(checkbox1.getAttribute("checked").equals("true"));
```

Un simple `click()` sobre el checkbox el marca o el desmarca, igual que faria l'usuari. Després tornem a llegir l'atribut `checked` per confirmar que el canvi s'ha produït.

___

### 📊 Resum: com treballar amb checkboxes

| Acció | Codi |
|--------------------------------|--------------------------------------------------------|
| Verificar que està **marcat** | `assertTrue(cb.getAttribute("checked").equals("true"));` |
| Verificar que està **desmarcat** | `assertNull(cb.getAttribute("checked"));` |
| Alternativa amb `isSelected()` | `assertTrue(cb.isSelected());` |
| **Marcar** o **desmarcar** | `cb.click();` |
| Marcar només si no està marcat | `if (!cb.isSelected()) cb.click();` |
