# KeyboardKeysTest.java - Simulació de teclat amb Keys i Actions 

---
Selenium permet simular pulsacions de tecles especials (Enter, Tab, fletxes, Escape…) mitjançant l'enumeració `Keys` i la classe `Actions`. Això és especialment útil per interactuar amb dreceres de teclat, navegar per formularis o simular comportaments que no es poden fer només amb `click()`. Aquest test treballa contra [the-internet.herokuapp.com/key\_presses](http://the-internet.herokuapp.com/key_presses).

___

### 📄 Codi del test

```java
KeyboardKeysTest.java — Simulació de teclat amb Keys i Actionspublic class KeyboardKeysTest extends BaseTest {

    @Test
    public void KeyboardKeysExample() throws InterruptedException {

        driver.get("http://the-internet.herokuapp.com/key_presses");

        // Crear l'objecte Actions vinculat al driver
        Actions builder = new Actions(driver);

        // Prémer la tecla ESPAI
        builder.sendKeys(Keys.SPACE).build().perform();
        Thread.sleep(2000);
        assertTrue(driver.findElement(By.id("result"))
            .getText().equalsIgnoreCase("You entered: SPACE"));

        // Prémer la tecla fletxa ESQUERRA
        builder.sendKeys(Keys.LEFT).build().perform();
        Thread.sleep(2000);
        assertTrue(driver.findElement(By.id("result"))
            .getText().equalsIgnoreCase("You entered: LEFT"));
    }
}
```

___

### 🔍 Anàlisi del test

#### La classe `Actions`

`Actions` és el constructor d'accions avançades d'usuari a Selenium. Permet encadenar múltiples accions (moure el ratolí, prémer tecles, fer clic) i executar-les totes juntes com una seqüència.

```java
Actions builder = new Actions(driver);

// Estructura general:
builder
  .accio1()
  .accio2()
  .build()    // Construeix la seqüència d'accions
  .perform(); // L'executa al navegador
```

___

#### L'enumeració `Keys`

`Keys` conté constants per a totes les tecles especials del teclat. S'utilitza tant amb `Actions` com directament amb `sendKeys()` en un element.

| Constant `Keys` | Tecla | Ús típic |
|-------------------------------|------------|-----------------------------------|
| `Keys.ENTER` | ↵ Enter | Enviar formularis, confirmar |
| `Keys.TAB` | ⇥ Tab | Navegar entre camps de formulari |
| `Keys.ESCAPE` | Esc | Tancar modals, cancel·lar |
| `Keys.SPACE` | Espai | Marcar checkboxes, activar botons |
| `Keys.BACK_SPACE` | ⌫ Retrocés | Esborrar caràcters |
| `Keys.DELETE` | Supr | Esborrar text seleccionat |
| `Keys.LEFT / RIGHT / UP / DOWN` | ← → ↑ ↓ | Navegar per llistes, sliders |
| `Keys.HOME / Keys.END` | Inici / Fi | Anar al principi/final d'un camp |
| `Keys.CONTROL + "a"` | Ctrl + A | Seleccionar tot |
| `Keys.F5` | F5 | Recarregar pàgina |

___

### 🧰 Formes d'utilitzar `Keys`

#### 1\. Amb `Actions` — sobre el focus actual de la pàgina

```java
Actions builder = new Actions(driver);
builder.sendKeys(Keys.ESCAPE).build().perform();
```

#### 2\. Amb `sendKeys()` — sobre un element concret

```java
WebElement camp = driver.findElement(By.id("searchbox_input"));
camp.sendKeys("text" + Keys.ENTER); // Escriu i prem Enter
```

#### 3\. Combinacions de tecles (ex: Ctrl+A, Ctrl+C)

```java
Actions builder = new Actions(driver);

// Seleccionar tot el text d'un camp
builder.keyDown(Keys.CONTROL)
       .sendKeys("a")
       .keyUp(Keys.CONTROL)
       .build()
       .perform();
```

___

> 💡 **Tip:** A macOS, utilitza `Keys.COMMAND` en lloc de `Keys.CONTROL` per a les dreceres de teclat (⌘ en comptes de Ctrl).
